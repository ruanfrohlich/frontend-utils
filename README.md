### 🏠 [Homepage](https://github.com/ruanfrohlich/frontend-utils)

## Prerequisites

- node >= 20
- yarn >= 1.22.22

## Install dependencies

```sh
yarn install
```

## Usage

```sh
yarn docs
```

Will open storybook
